Welcome to WaveMaker User Management Template Application

This application demonstrates how to build a User Management System using WaveMaker.

The application is hosted on: http://localhost:8094/UserManagement

Features include:

  • Self-service new account registration with integrated email workflow
  • Automated email verification
  • Lost or forgotten password management with password reset email
  • MD5 encryption of passwords
  • Email and password validation using regular expressions
  • Remember username using cookie
